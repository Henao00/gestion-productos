Sobre UniWeb

Sistema de gestión de inventario Django · Python · v3.0

¿Qué es UniWeb?

UniWeb es una plataforma web de gestión de inventario desarrollada con Django y Python. Permite registrar, editar, buscar y eliminar productos de forma rápida e intuitiva, con una interfaz moderna diseñada para equipos que necesitan control total sobre su catálogo.

El proyecto nació como una solución práctica para centralizar la gestión de productos en un solo lugar, eliminando hojas de cálculo y procesos manuales.

Historia

Evolución del proyecto

v1.0
Primer prototipo
CRUD básico de productos con Django. Formularios simples, tabla HTML y estilos mínimos. Funcional pero sin identidad visual.
v2.0
Mejora de UI
Rediseño con tipografía editorial (Fraunces + IBM Plex Mono), sidebar oscuro, sistema de badges y estadísticas básicas.
v3.0 · Actual
Rediseño SaaS premium
Arquitectura de páginas separadas (Home, Nosotros, Inventario, Subir). Diseño oscuro estilo Vercel/Linear con gradientes, glassmorphism, animaciones y responsive completo.
